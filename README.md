# frameworkcss

Instalar node js

Testar node - node --version

Testar NPM - npm --version

Instalar SASS

Abrir VS Code e procurar pela extensão live sass compiler 

Crie uma pasta onde vai ter os arquivos SASS, para testar, terá uma opção no formato de olho, se estiver tudo certo, vai aparecer watching, indicando que está pronto para usar.
